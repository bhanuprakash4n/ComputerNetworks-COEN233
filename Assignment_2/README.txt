Programming Assignment 2: Client using customized protocol on top of UDP protocol for rquesting identification from server for access permission to the cellular network.
One client connects to one server.

Steps to run:
1) Open terminal and compile the server-2.c file using "gcc -o server-2 server-2.c"     
2) Open another terminal and compile the client-2.c file using "gcc -o client-2 client-2.c"
3) Run the server first by using "./server-2"
4) Run the client using "./client-2" it will begin transmitting packets to the server-2.
